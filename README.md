# bookii
Book sales management

## Getting Started
install dependencies: `npm install`  
run app: `npm start`

### Deploying

* `npm install -g surge`
* `npm run build`
* `mv dist/index.html dist/200.html`
* `surge dist your-project-name.surge.sh`
